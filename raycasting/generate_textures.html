Dungeon Game Texture Generator

This page will generate placeholder textures for the Dungeon Adventure game. Click the button below to generate all textures. The browser will prompt you to download each texture file.

Important: After downloading each texture, make sure to save it to the correct directory in the game's assets folder:

  • Wall textures (stone_wall.png, brick_wall.png, wood_wall.png) → assets/textures/
  • Door textures (door_closed.png, door_open.png) → assets/textures/
  • Sprites (player_hand.png, skeleton_idle.png, etc.) → assets/sprites/

Textures that will be generated:

  • Wall Textures: stone_wall.png, brick_wall.png, wood_wall.png
  • Door Textures: door_closed.png, door_open.png
  • Sprites: player_hand.png, skeleton_idle.png, health_potion.png, key_gold.png, chest_closed.png, chest_open.png
